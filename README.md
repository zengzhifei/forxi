## 核心功能

### 📄 文件预览

支持 PDF、Word、Excel、PPT、TXT、图片、音视频等常见文件格式，无需安装对应软件，直接在浏览器中预览。拖拽上传或输入 URL 远程预览，随心所欲。

**适用场景**：临时需要查看一份文档、收到客户发来的特殊格式文件、分享文件前快速确认内容、下载前先预览内容是否正确。

---

### 🖼️ 图片处理

集合 14+ 种日常图片处理功能，涵盖：

**基础处理**：图片压缩、裁剪、旋转、缩放

**创意编辑**：添加水印、添加文字、图片拼接、九宫格切割、圆形裁剪

**视觉效果**：图片滤镜（灰度、复古、提亮、对比、模糊）、马赛克

**格式转换**：JPG / PNG / WebP 格式互转

**AI 智能处理**：AI 抠图（生成透明前景图）、AI 背景透明（移除背景）、AI 智能裁剪（识别主体）、AI 证件照（生成标准证件照）

纯本地处理，文件不离手。

**适用场景**：朋友圈配图需要压缩、身份证复印件需要加水印、产品图需要统一尺寸、证件照需要换底色、需要将图片切割成九宫格发朋友圈、需要去除图片背景。

---

### 📝 图片 OCR

一键提取图片中的文字内容，支持中文、英文、日文、韩文等多种语言识别。纯前端处理，文件不上传服务器，保护隐私。

**适用场景**：截图文字提取、扫描件电子化、证件信息录入、名片信息整理、PDF 图片文字复制。

---

### 🔄 文件转换

支持将多种格式转换为 PDF：

**图片转 PDF**：将 JPG、PNG、WebP 等图片格式转换为 PDF 文档

**文本转 PDF**：纯文本或 Markdown 内容直接转为 PDF

**Word 转 PDF**：上传 Word 文档，转换为 PDF 格式

**表格转 PDF**：Excel 表格数据转为 PDF

适合制作报告、归档资料等场景，无需安装软件，纯浏览器端处理。

---

### 📊 流程工厂

可视化流程图编辑器，让流程图制作变得简单高效。

**核心功能**：拖拽添加节点、连接线自动吸附、节点属性编辑、批量节点操作

**支持多种节点类型**：开始/结束、流程、判定、数据、输入/输出、准备

**导出能力**：支持导出 PNG、JPEG 等多种图片格式，方便嵌入文档或演示文稿

**适用场景**：业务流程梳理、系统架构设计、算法流程说明、项目方案展示。

---

### 🤖 AI 创作

AI 驱动的创意工具，让想象变成现实。

**文生图**：输入一段文字描述，AI 在几秒内为你生成对应的图片。无需绘画技能，只需想象力。支持写实摄影、动漫插画等多种风格可选。

**图生图**：上传一张参考图，用文字描述你想要的改变，AI 帮你实现。可以是风格的转换、元素的添加、画面的优化。

**AI 对话**：与多个 AI 模型对话，解答问题、辅助写作、编写代码、头脑风暴。输入你的想法，AI 帮你整理成文字。

**适用场景**：社交媒体配图、内容创作素材、产品原型示意、灵感收集与可视化、文章配图、PPT 素材。

---

### 🔧 实用工具箱

为开发者和日常办公人群精选的效率工具集合。

**编码转换**：Base64、URL 编码、HTML 实体、进制转换（二进制/八进制/十进制/十六进制）、颜色值互转（HEX/RGB/HSL）

**格式处理**：JSON 格式化与压缩、JSON 与 Java 对象互转、JSON 与 Go 结构体互转、JSON 与 YAML 互转、SQL 语句格式化、Markdown 转 HTML

**生成器**：UUID 生成、随机字符串生成、二维码生成（支持文本/网址/联系方式）、HMAC 哈希、文本哈希（MD5/SHA-1/SHA-256）、Lorem Ipsum 占位文本生成、Crontab 表达式生成

**测试工具**：正则表达式测试与验证、密码强度检测

**解析工具**：JWT Token 解析与验证、User Agent 解析、URL 解析与参数提取

**实用查询**：HTTP 状态码参考、MIME 类型参考、NATO 字母表

**文本工具**：字符统计与分析、大小写转换（驼峰/帕斯卡/ kebab/下划线）、文本反转、Slugify 生成、字母排序、列表转其他格式

**加密工具**：文本对称加密与解密

**其他工具**：温度单位转换（摄氏度/华氏度/开尔文）、IPv4 地址转换、IPv4 子网计算器

**适用场景**：前后端联调时格式化 JSON 数据、生成唯一标识符、测试正则表达式、转换颜色值格式、生成测试用占位文本、验证 JWT Token 内容。

## 设计理念

**少即是多。** 每一个功能都经过精心设计，力求用最少的操作步骤完成最核心的需求。我们讨厌臃肿的功能堆砌，更相信把一件事做到极致。

**工具应该透明。** 当你需要文件预览时，不该被要求注册账号。当你想处理图片时，不该被迫观看三十秒广告。Forxi 存在的意义就是帮你完成任务，然后安静地退到一边。
