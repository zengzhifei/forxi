# Forxi

> 探索无限可能，创造美好未来

Forxi 是一个集文件预览、图片处理、AI 工具、IT 工具于一体的多功能平台，让你的工作更加高效便捷。

## 功能介绍

### 📁 文件预览
- 支持 PDF、Word、Excel、PPT、TXT、代码文件等多种格式
- 在线直接预览，无需下载
- 响应式设计，支持各种设备

### 🎨 AI 趣玩
- 在线文生图：根据文字描述生成精美图片
- 多种模型选择
- 多种尺寸比例可选

### 🖼️ 图片处理

| 功能 | 说明 |
|------|------|
| 图片压缩 | 调整压缩质量，减小文件大小 |
| 图片旋转 | 任意角度旋转 |
| 图片缩放 | 调整尺寸大小 |
| 添加水印 | 文字/图片水印 |
| 添加文字 | 在图片上添加文字 |
| 图片拼接 | 多图横向/纵向拼接 |
| 九宫格 | 将图片切割为九宫格 |
| 圆形裁剪 | 将图片裁剪为圆形 |
| 证件照 | 常用证件照尺寸 |
| 滤镜效果 | 灰度、复古、提亮、对比、模糊等 |
