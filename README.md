# Forxi

> 皆可成为自己的指路明灯

Forxi 是一个免费在线工具平台，提供文件预览、图片处理、AI 创作、IT 开发工具箱等实用功能，无需安装，打开即用。

🌐 **https://forxi.cn**

---

## 功能模块

### 📄 文件预览

无需安装任何软件，直接在线预览各类文件。

- 支持 PDF、Word、Excel、PPT、TXT、代码文件等多种格式
- 支持本地上传或远程 URL 预览
- 响应式设计，移动端友好

### 🖼️ 图片处理

纯本地处理，文件不上传服务器，保护隐私。

| 功能 | 说明 |
|------|------|
| 图片压缩 | 调整质量，减小文件体积 |
| 图片裁剪 | 自由裁剪或圆形裁剪 |
| 图片旋转 | 任意角度旋转 |
| 图片缩放 | 自定义尺寸大小 |
| 添加水印 | 文字 / 图片水印 |
| 添加文字 | 自定义字体与位置 |
| 图片拼接 | 多图横向 / 纵向拼接 |
| 九宫格切割 | 一键切割为九宫格 |
| 证件照制作 | 常用证件照尺寸 |
| 滤镜效果 | 灰度、复古、提亮、对比、模糊等 |
| 格式转换 | JPG / PNG / WebP 互转 |

### 🤖 AI 趣玩

- 文生图：根据文字描述生成图片
- 图生图：基于参考图进行 AI 重绘
- AI 对话：多模型智能对话

### 🔧 IT 工具箱

开发者日常用得上的小工具，一站搞定。

**JSON**：格式化 / 压缩 / 对比 / 转 Java / 转 Go / 转 YAML

**转换**：Base64、颜色（HEX/RGB/HSL）、进制、温度、IPv4

**编码**：URL 编码解码、HTML 实体编码解码

**生成**：UUID、随机字符串、二维码、HMAC 哈希、文本哈希、Crontab 表达式、Lorem Ipsum 占位文本

**分析**：密码强度检测

**测试**：正则表达式测试

**解析**：JWT Token、User Agent、URL

**文本**：字符统计、大小写转换、Markdown 转 HTML、Slugify、NATO 字母、文本反转、SQL 格式化、列表转换、字母排序

**参考**：HTTP 状态码、MIME 类型

**加密**：文本加密 / 解密

### 🧲 JetBrains 激活

支持 IntelliJ IDEA、PyCharm、WebStorm、GoLand、Rider、CLion 等 JetBrains 系列 IDE。

---

## 技术栈

- **前端**：Vue 3 + Vite + Tailwind CSS
- **路由**：Vue Router
- **部署**：静态托管（CSR SPA）

---

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
